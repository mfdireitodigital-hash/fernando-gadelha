# Cartilha de Segurança para Influenciadores

## Fernando Gadelha (@feehzero1)

Esta cartilha foi criada especialmente para você, Fernando. Ela ensina tudo sobre os gatilhos que fazem a Meta bloquear contas de influenciadores, com foco em cassinos, rifas e conteúdo de rua.

### Conteúdo

- Introdução e contexto
- Os 5 gatilhos principais da Meta
- Cassinos: por que são tão perigosos
- Rifas e sorteios: a zona cinzenta
- Hashtags perigosas e seguras
- Imagens que acionam bloqueios
- Casos reais de influenciadores bloqueados
- Dicas práticas para conteúdo de rua
- Checklist pré-publicação
- Quiz interativo

### Como Usar

Abra o arquivo `index.html` em seu navegador. A cartilha é responsiva e funciona em desktop, tablet e mobile.

### Versões

- `index.html` - Versão Web (desktop)
- `index_mobile.html` - Versão Mobile (otimizada para celulares)
